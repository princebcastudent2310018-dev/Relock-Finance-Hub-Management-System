# Relock Finance Management System

## 📌 About
This project is a finance management system to track expenses and income.

## 🚀 Features
- Expense tracking
- Income management
- Reports generation

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- MySQL

## ▶️ How to Run
1. Download project
2. Setup database
3. Run on localhost# Relock-Finance-Hub-Management-System
Relock Finance Management System is a web-based application designed to help users efficiently track and manage their income and expenses. It provides a simple interface for financial monitoring, report generation, and better money management.
